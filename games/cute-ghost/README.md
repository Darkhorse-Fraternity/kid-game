# 阿飘来了 (Cute Ghost)

> 一款可爱风格的多人生存游戏

## 游戏简介

夜幕降临，一只可爱的小阿飘出现在宿舍楼！小伙伴们纷纷躲进各个房间，希望能逃过阿飘的"拜访"。阿飘会逐个敲门，攻破房间找到里面躲藏的人。谁能坚持到最后，谁就是最终的幸存者！

## 游戏流程

### 1. 躲藏阶段
- 游戏开始时，**8 个 NPC** 会随机选择房间躲藏
- 每个房间最多容纳 **2 人**
- 有些房间可能没人，有些房间挤满了人

### 2. 阿飘巡逻
- 阿飘在走廊中漂浮，寻找目标房间
- 阿飘会随机或按策略选择房间攻破

### 3. 攻破房间
- 阿飘飘到房间门口，开始敲门
- 敲门动画后，房间门被打开
- 房间里的所有人被阿飘"带走" 💀

### 4. 继续狩猎
- 阿飘继续选择下一个房间
- 直到只剩最后的幸存者

## 游戏机制

### 👻 阿飘行为
- 在走廊中缓缓漂浮
- 随机选择房间进行攻破
- 每次攻破需要一定时间（敲门动画）
- 可能会假装离开再折返

### 🚪 房间状态
| 状态 | 说明 |
|------|------|
| 🚪 关闭 | 房间完好，可能有人躲藏 |
| 🚪💀 已攻破 | 阿飘已光顾，里面的人已被带走 |
| 🚪✨ 空房间 | 没有人躲藏的房间 |

### 👥 NPC 状态
| 图标 | 状态 |
|------|------|
| 😰 | 躲藏中（紧张） |
| 😨 | 阿飘接近（害怕） |
| 💀 | 被带走了 |
| 🎉 | 幸存者！ |

## 胜利条件 

🏆 **最后存活的 NPC 获得胜利！**

- 如果只剩 1 人存活 → 该 NPC 获胜
- 如果最后一个房间有多人 → 他们共同获胜

## 游戏特色

### 可爱画风
- 圆润可爱的小阿飘设计
- 明亮温馨的宿舍配色
- 夸张有趣的表情动画

### 紧张刺激
- 不知道阿飘会选哪个房间
- 房间里人越多越容易被发现？
- 空房间的策略：会被跳过还是浪费阿飘时间？

### 随机性
- NPC 随机分配房间
- 阿飘随机选择目标
- 每局游戏都不一样

## 技术信息

- **游戏引擎**: Phaser 3
- **开发语言**: TypeScript
- **端口**: 3003

## 运行游戏

```bash
# 进入游戏目录
cd games/cute-ghost

# 安装依赖
bun install

# 启动开发服务器
bun dev
```

游戏将在 http://localhost:3003 打开
