# 🎮 游戏乐园 - Game Learn

**和 AI 一起创造属于你的游戏世界！**

这是一个为小朋友设计的游戏编程学习项目。不需要按部就班学习，直接告诉 AI 你想做什么游戏，它会帮你写代码！

---

## 🚀 快速开始

### 1. 安装依赖

```bash
# 确保你已经安装了 Bun (https://bun.sh)
bun install
```

### 2. 启动游戏展示页

```bash
bun dev
```

浏览器会自动打开，显示所有可玩的游戏！

### 3. 运行单个游戏

```bash
# 运行接星星游戏
cd games/catch-the-stars && bun dev

# 或者运行 Flappy Bird
cd games/flappy-bird && bun dev
```

---

## 🎯 创建你自己的游戏

### 方法一：复制模板

```bash
# 创建一个叫 my-cool-game 的新游戏
bun run new my-cool-game

# 进入游戏目录并启动
cd games/my-cool-game && bun dev
```

### 方法二：让 AI 帮你写

打开 `games/你的游戏/src/main.ts`，然后告诉 AI（比如 Claude 或 ChatGPT）：

> 我在用 KAPLAY 游戏框架，帮我写一个 [描述你想要的游戏]。
> 要求：
> - 完整可运行的代码
> - 有键盘或鼠标控制
> - 有简单的游戏目标

把 AI 生成的代码复制到 `main.ts` 文件，保存后立刻就能看到效果！

## 🎮 已有的示例游戏

### ⭐ 接星星 (catch-the-stars)
- **玩法**：用键盘左右方向键移动篮子，接住掉落的星星
- **难度**：⭐ 简单
- **学到**：移动控制、碰撞检测、计分系统

### 🐦 Flappy Bird
- **玩法**：按空格键或点击让小鸟飞起来，穿过管道
- **难度**：⭐⭐ 中等
- **学到**：重力物理、障碍物生成、游戏状态管理

## 💡 和 AI 对话的技巧

### 创建新游戏
```
我在用 KAPLAY 游戏框架，帮我写一个打砖块游戏。
要求：
- 一个可以左右移动的挡板
- 一个会弹跳的球
- 上方有一排砖块，球碰到砖块就消除
- 有分数显示
```

### 修改游戏
```
这是我的游戏代码：
[粘贴你的代码]

我想要：让小球速度变快一点
```

### 修复错误
```
我的游戏代码报错了：
[错误信息]

代码是：
[粘贴代码]

帮我修复。
```
